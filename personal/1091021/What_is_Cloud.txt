클라우드 컴퓨팅은 쉽게 인터넷 서버를 통해서, 네트워크나 여러 컨텐츠 사용,
그리고 데이터를 저장하는 거나 여러가지 북치고 장구치고 이런 식으로 다하는 거다.

IaaS 는 가상화된 환경이라고 할 수 있다. 직접 못 만들었을 때,
가상의 서버에 서비스를 구축하고 관리도 하는 것.

PaaS 는 플랫폼 중에서도 표준화가 된 플랫폼을 서비스하는 것이다. 
물론 가상화가 된 소프트웨어, 하드웨어 등등을 필요에 따라 제공을 하며,
개발 관련 환경이나 프로세스 등을 제공하는 것이다.

SaaS 는 소프트웨어를 다 설치하는 게 아닌, 필요한 걸 사용자가 골라가는 걸 말한다.
요즘은 서버에서 작동되어서 네트워크를 통해 소프트웨어의 일부 기능을 온라인에서 이용하는
방식이 쓰인다. 대부분이 필요없는 것까지 함께 있는 걸 산다. 
쉽게 생선이라 치면, 한 마리 전체가 만원이고, 몸통은 5 천원이라 치자.
그럼 몸통만 사는 게 SaaS 고, 전체 다 사는 게 대부분의 소프트웨어다.
ASP 서비스와 비슷하다. 
( Application Service Providing . 앱 서비스를 빌려주는 것. )
( Armyment Supply Point 가 아니다. )

BaaS 는 모바일 플랫폼이라 한다.
크게 두 가지로 나눌 수 있는데, 흔히 컴퓨터에서 쓰는 Windows 8.1 같은 OS 플랫폼 
( 운영체제 플랫폼 )
그리고, 페이스북, 트위터, 애니팡, 카카오톡, 다함께 차차차같은 컨텐츠를 제공하는
서비스 플랫폼 두 가지가 있다.
현재 안드로이드가 거의 80 % 가까운 점유율을 보이고 있기 때문에,
안드로이드 관련 플랫폼이나 앱 개발이 대세다.

클라우드 컴퓨팅의 예를 들자면, 집에 아무도 없을 때, 난방을 꺼주거나,
수소가스 밸브를 잠궈주던가, 집에 올 때쯤에 난방을 틀어주고, 차가운 목욕물을 받아준다던가
하는 그런 기능을 보이는 것이 쉬운 예라 할 수 있겠다.
